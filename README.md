# Data Storytelling with Amazon Sales

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/sohailblockchain/week-3-classs-Telling-a-Story-with-Data/blob/main/amazon_data_storytelling_lab.ipynb)

A beginner-friendly Data Science lab for transforming sales data into a clear, evidence-based business story using pandas, Matplotlib, and seaborn.

## Skills covered

- Dataset validation and date preparation
- KPI calculation and category aggregation
- Bar, line, and scatter-chart selection
- Sales–profit correlation
- Evidence-based insight and recommendation writing
- Communicating limitations responsibly
